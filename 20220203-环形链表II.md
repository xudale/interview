# 20220203-环形链表

给你一个链表的头节点 head ，判断链表中是否有环。

链接：https://leetcode-cn.com/problems/linked-list-cycle

双指针。

```JavaScript
var hasCycle = function(head) {
    let slow = fast = head
    while(fast?.next?.next) {
        slow = slow.next
        fast = fast.next.next
        if (slow == fast) {
            return true
        }
    }
    return false
};
```
